# 도서관 홈페이지 팀 프로젝트

## 개요

> 부산 IT 교육센터에서 진행한 팀 프로젝트로, 6명이 한 팀이 되어 작업했습니다. 도서를 대출/예약할 수 있고, 열람실을 이용할 수 있는 가상의 도서관을 상상하며 홈페이지를 만들었습니다.
